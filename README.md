# BookVerse - Digital Library & Bookstore

A modern digital bookstore platform where users can purchase or borrow books. Built with React, Node.js, Express, and Supabase.

## Tech Stack
- **Frontend**: React + Vite, Tailwind CSS, i18next (localization)
- **Backend**: Node.js, Express
- **Database**: Supabase (PostgreSQL)
- **Authentication**: JWT & Supabase Auth

## Features
- Dynamic Book Catalog
- Multi-language support (AR/EN)
- Real-time shopping cart
- Digital Book Reader (ePub/PDF)
- Admin Dashboard for inventory management
- User Library & Favorites

## Setup
1. Clone the repo
2. Run `npm install` in both `/frontend` and `/backend`
3. Set up your `.env` variables (Supabase URL, Key, etc.)
4. Run `npm run dev` in both directories
